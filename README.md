# HTML & CSS Course Projects

This repository contains all the projects I completed during the HTML and CSS course.  
Each project was created to practice and reinforce fundamental front-end development skills.
https://flynnzin.github.io/Portfolio-Website/


## Projects

### 1. Basic HTML Structure
A project focused on understanding HTML structure, semantic tags, headings, paragraphs, lists, and links.

### 2. CSS Styling Basics
This project demonstrates the use of CSS for styling, including colors, fonts, spacing, and basic layout techniques.

### 3. Layout and Box Model
A project created to practice the CSS box model, margins, padding, borders, and element positioning.

### 4. Responsive Design
This project focuses on building responsive layouts using flexible units and basic responsive design principles.

### 5. Portfolio Website
A personal portfolio website created using only HTML and CSS, showcasing my projects and skills.  
This project is deployed live using GitHub Pages.
